Modern Router Features ⚡

Powerful utilities for Standalone components.

withComponentInputBinding()

Simple Idea:

URL data → directly goes into @Input()

/product/101?filter=active
⬇️
@Input() id!: string; // 101 @Input() filter?: string; // active
Implementation:
// 1. Enable in app.config.ts
provideRouter(routes, withComponentInputBinding())

// 2. Use in your component
@Input() id!: string;

withInMemoryScrolling()

Simple Idea:

Remembers where you scrolled and handles anchor links.

  • Enabled: Back button → Old position
  • Anchor: /page#section → auto scroll
Implementation:
provideRouter(routes, withInMemoryScrolling({{ '{' }}
  scrollPositionRestoration: 'enabled',
  anchorScrolling: 'enabled'
{{ '}' }}))
🎯 What Happens?
🔹 1. Scroll Restore
  • Go to /page
  • Scroll down half-way
  • Navigate away (e.g., to /about)
  • Click browser BACK button
  • 👉 Scroll position is restored automatically
🔹 2. Anchor Scrolling

Link: <a routerLink="/page#section2">Go to Section 2</a>

👉 Page automatically scrolls to the ID matched section

🔥 Final Simple Difference

Feature Core Use Case
withViewTransitions() ✨ Smooth page animations (fade/slide)
withInMemoryScrolling() 🖱️ Precise scroll position control

Putting it All Together 🛠️

How to declare your routes and navigate between them.

1. Route Declaration

Define your path segments as usual. Angular will handle the binding automatically.

// app.routes.ts
export const routes: Routes = [
  {{ '{' }} 
    path: 'product/:productId', 
    component: ProductComponent 
  {{ '}' }}
];

2. Navigation

Navigate with parameters. The value '101' will be passed as 'productId'.

<!-- Declarative -->
<a routerLink="/product/101">View Product</a>

// Imperative
this.router.navigate(['/product', 101]);

3. Component Usage

Simply use the @Input() with the SAME NAME as your path parameter.

@Component(...)
export class ProductComponent {{ '{' }}
  // Matches :productId in route
  @Input() productId!: string; 
{{ '}' }}
💡 WHY IT MATTERS: These features reduce boilerplate code and make your app feel faster and more professional with minimal configuration.
